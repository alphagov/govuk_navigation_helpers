# GOV.UK Navigation Helpers

This is a gem to share code between GOV.UK frontends.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'govuk_navigation_helpers', '~> VERSION'
```

And then execute:

    $ bundle
